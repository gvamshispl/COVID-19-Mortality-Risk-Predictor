# COVID-19 Mortality Risk Prediction System

This project uses **Machine Learning** to predict the **mortality risk of COVID-19 patients** based on their medical information.  
The goal is to help identify **high-risk patients early** so that healthcare professionals can make faster decisions.

---

## Project Features

- Predicts patient **mortality risk**
- Uses **Random Forest Machine Learning algorithm**
- Handles **imbalanced data using SMOTE**
- Performs **data preprocessing and feature selection**
- Provides **probability of survival and death**
- Deployed as a **Streamlit web application**

---

## Input Features

The model uses important patient information such as:

- Age  
- Pneumonia  
- Diabetes  
- Hypertension  
- Obesity  
- ICU admission  
- Intubation status  
- COVID test result
